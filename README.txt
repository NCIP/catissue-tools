caGrid 1.0 SDK 3.2 Query Processor

This project contains code for the implementation of a CQL Query Processor against a caCORE SDK 3.2 data source.

This project requires Java 1.5 to build as it depends on the caCORE SDK's client libraries, which are built with Java 1.5

CSM security in this implementation is provided by the API of the authz project.
